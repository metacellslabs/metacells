# Sponsorship

MetaCells is an open project developed in public.

The project explores computational models for automation, AI workflows, and system design based on a cell-oriented architecture. Development of such infrastructure requires continuous research, experimentation, and engineering effort.

Sponsorship helps sustain this work and allows the project to evolve as open infrastructure.

---

# Why support the project

Supporting MetaCells helps fund:

- research and experimentation
- development of the core architecture
- documentation and examples
- infrastructure and maintenance
- long-term stability of the project

The goal is to keep MetaCells open and accessible for developers, researchers, and organizations.

# Organizations

Companies and research groups using MetaCells for modeling, experimentation, or internal systems are encouraged to support the project.

Organizational sponsorship helps prioritize:

- stability and maintenance
- documentation and tutorials
- ecosystem tooling
- research features

If your organization is interested in supporting the project or collaborating on research, please contact the maintainer.

---

# Other ways to contribute

Financial support is not the only way to help the project grow.

You can also contribute by:

- improving documentation
- proposing ideas
- reporting issues
- contributing code
- creating examples and models
- sharing the project
